Simple light weight plugin that automatically makes all currently and future embedded video's responsive.  There is no setup required, it just works.  Currently Youtube and Vimeo are supported with more to come.

== Description ==

Simple light weight plugin that automatically and recursively makes all YouTube and Vimeo video's site wide responsive.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to a page with a video and shrink the window and watch the magic!

== Frequently Asked Questions ==

= My videos aren't resizing?  =

This plugin will resize the video's to the width of its container, meaning that if your entire site is wider than the screen width then the video will stay wide.  If you have a detailed question about this please feel free to contact me via http://www.brandonorndorff.com.

= Where do I ask questions or leave comments?  =

If you have any questions please visit [WP Responsive Video Plugin Page](http://www.brandonorndorff.com/wordpress/plugins/wp-responsive-video-wordpress-plugin/ "WP Responsive Video") and leave them in the comments.

== Demo ==

Go to: http://www.brandonorndorff.com/wordpress/plugins/aas-digg-digg-alternative-wordpress-plugin/ and scroll to the bottom and shrink your browser window and watch the video resize to fit the screen.
